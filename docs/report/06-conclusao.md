# 6. Conclusão

O MedFlow demonstra um MVP funcional do SGHSS com stack moderna (Next.js full-stack leve), atendendo requisitos prioritários da VidaPlus em prazo acadêmico.

## Lições aprendidas

- Server Actions simplificam formulários sem API REST explícita
- Auth.js com JWT atende controle de perfis sem OAuth complexo
- Testes e2e com Playwright validam fluxos críticos rapidamente
